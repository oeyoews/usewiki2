## Usewiki2

> [!IMPORTANT]
> 此插件目前支持 google chrome, Edge, 暂不支持 Firefox.

> [!TIP]
> UseWiki2: usewiki 的 vue 版本，但是带来了更多功能。

> 将当前网页文章转换成 Markdown 文件，支持保存到本地，TiddlyWiki5, 支持 AI 整理润色。
